# Dotnet.Bacon
A dotnet tool to display tasty bacon

## Compilation and execution
```
dotnet build
dotnet run
```

## Download and install latest from nuget
```
dotnet tool install --global bacon
```

## Usage:
```
dotnet bacon
```
With eggs
```
dotnet bacon egg
```
